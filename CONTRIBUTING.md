# Contributing

Thanks for your interest in improving DXF File Viewer.

## Contributor License Agreement (CLA)

Before we can accept pull requests, contributors must sign the project CLA via cla-assistant on GitHub.

By signing the CLA, you confirm that you have the right to submit your contribution and grant the project maintainer the right to relicense contributions in future versions of the project.

## Pull request basics

- Keep changes focused and small.
- Describe the user-facing impact in the PR description.
- Include reproduction and verification steps when fixing bugs.
