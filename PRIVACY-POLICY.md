# Privacy Policy for DXF File Viewer

DXF File Viewer is a read-only file viewer extension.

## Data handling

- The extension does not require account creation.
- The extension does not intentionally collect personal information.
- DXF files are opened for viewing functionality only.

## Network and third parties

- No backend service is required for the core MVP behavior.
- No analytics provider is configured in this MVP.

## Changes

This policy may be updated as the product evolves. The latest version of this file is the active policy.

## Contact

For questions, use the support contact listed in the Chrome Web Store entry.
